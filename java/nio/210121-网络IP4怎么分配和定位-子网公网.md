1. 子网掩码(255.255.255.0) 说明这个子网内可以有255台机器, (1111111111111111-1111111-00000000) 后面的0就是自己可以有的服务器.
2. IPV4不够用, 每个没有公网IP的机器都有自己的子网ip. 只要IP在几个路段内都是子网IP: (172.16.xx-172.31.xx, 10.xx, 192.168.xx). 127.0.0.1就是本机.
